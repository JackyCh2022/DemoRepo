У якості способи мотивації для дітей при викладанні курса програмування на python можливо використовувати наступне:

  1. Використання високоякісних презентацій для проведення занять, з красивими фото, картинками, примерами
  2. Обговорення з учнями людських особливостей відомих розробників, авторів основних проектів, їх досягнень і провалів.
  3. Використання ігрових завдань у зв'язку з вирішенням програмістських завдань. Наприклад:
    • при вивченні оператора if - гра "Вгадай число"
    • при вивченні циклу while - гра "Купи слона"
    • при вивченні циклу for - гра з розрахунком кількості бактерій або вірусів
  4. Використання яскравих кольорових рішень - багатокольорові завдання при вивченні графічного інтерфейсу користувача, використання графіки turtle
  5. На досить продвинутому рівні учнів - розбор і реалізація реальних ігор з використанням відомих бібліотек (для python, наприклад, pygame)
